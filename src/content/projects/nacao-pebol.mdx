# Visão Geral do Projeto

**Nação Pebol** é um projeto de *sports* desenvolvido utilizando **Unity**. O jogo foi criado para a **GameJam+ 2020** na categoria **#FlaSports**.

O objetivo foi criar uma experiência de futebol *hypercasual* e única, focada na jornada com ídolos do Flamengo em busca da vitória.

## Desafios Técnicos e Plataformas

O principal desafio foi adaptar um jogo de futebol para uma experiência *hypercasual* com foco em **mobile** e **web** (HTML5), além de uma versão para **Windows**.

Minhas responsabilidades incluíram:

* **Desenvolvimento Multiplataforma:** Otimizar o código Unity para ser leve e responsivo em navegadores (HTML5) e dispositivos Android.
* **Mecânicas Simplificadas:** Criar a jogabilidade de futebol que fosse intuitiva e rápida, característica do gênero *hypercasual*.

## Tecnologias Utilizadas

O jogo foi desenvolvido em Unity, utilizando C# para as mecânicas de jogo.
